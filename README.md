# North Monaco Place

Property marketing website created for a residential land sale.

## Overview

North Monaco Place is a simple property presentation website built to market a residential development parcel and provide buyers with key information in a clean digital format.

The site was designed to support real-world property marketing efforts, including QR-code signage directing potential buyers to the listing.

## Features

- Property-focused landing page
- Mobile-friendly responsive design
- Zoning and development information
- Utility disclosure details
- External resource links for buyer due diligence
- QR-code compatible marketing workflow

## Purpose

This project demonstrates practical web product execution for a real business use case.

Goals included:

- quickly shipping a clean marketing site
- presenting structured real estate information clearly
- supporting physical/offline marketing with digital conversion via QR access
- creating a simple deployable product with minimal friction

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel deployment
